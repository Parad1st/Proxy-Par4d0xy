## 📋 Описание



**Прокси Парадокси** — это полностью бесплатный VPN-конфиг. 



✨ **Особенности:**

- 🌍 **Обычные сервера** — для доступа к зарубежным ресурсам

- 🛡️ **Белые списки (SNI-RU)** — для обхода блокировок российских сервисов через зарубежные домены

- 🔄 **Автообновление** — конфиг обновляется каждые 3 часа

- 🌐 **Мультиплатформа** — работает в Happ, v2rayTun, Shadowrocket, Hiddify и других клиентах
